### **Markdown основы**
- `# Заголовок 1`  
- `## Заголовок 2`  
- `### Заголовок 3`  
- `**жирный текст**`  
- `*курсив*`  
- `- Список` или `1. Нумерованный список`  
- Блок кода:
    ```markdown
    ```язык
    код
    ```
    ```
- Ссылки: `[текст ссылки](https://example.com)`  
- Изображения: `![альт текст](ссылка_на_картинку)`
